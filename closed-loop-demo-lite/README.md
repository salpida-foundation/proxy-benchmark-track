# Closed-Loop Demo-Lite

This folder contains public helper documents and placeholder code for a bounded local closed-loop demo-lite surface in the **SICS Human-State Proxy Benchmark Track**.

It exists to show how AI-output events, synthetic/sample proxy fields, dashboard review states, and feedback event logs may be connected as structure.

It is not the Sal-Meter core signal track.

It does not define CAIS.

It does not grant CAIS compliance.

It does not validate Sal-Meter.

It does not validate benchmark performance.

It does not introduce raw human data.

It does not create diagnostic, clinical, therapeutic, surveillance, employment, insurance, educational, legal, eligibility, certification, or human-ranking authority.

It does not deploy a live feedback loop.

It does not automate intervention on real people.

---

## Purpose

A closed loop is powerful.

A weak demo can look like a product.

A placeholder can look like intervention.

A review prompt can look like therapy.

A synthetic event log can look like proof.

This folder exists to prevent that drift.

The purpose of this folder is to define a safe public helper structure for a local, synthetic/sample-only demo-lite loop.

The loop may show how fields connect.

The loop must not claim that the system works on real people.

The loop may show structure.

The loop must not create authority.
