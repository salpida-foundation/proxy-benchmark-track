## Included helper layer

This pre-release includes the completed P3 public helper layer:

- Human-State-Aware AI Mediation helper layer
- Human-State Packet helper structure
- Dyadic Recovery Baseline Suite B0-B7
- Recovery Gate definition
- Termination Gate definition

## Core object distinction

The helper layer distinguishes:

```text
Human-State Packet
→ Dyadic Session Event
→ Benchmark Session Container
```

A Human-State Packet is a bounded state-summary helper object.

A Dyadic Session Event records public-safe boundary events.

A Benchmark Session Container records the public-safe benchmark container structure.

None of these objects validates the human body, Sal-Meter, CAIS compliance, benchmark success, clinical use, therapeutic use, counseling use, legal mediation authority, surveillance readiness, human ranking, relationship verdicts, or production intervention.
