# Data Boundary

This repository must not contain raw human data.

The Proxy Benchmark Track is a research-stage benchmark support track for synchronized multimodal human-state research. It is non-clinical, non-diagnostic, non-therapeutic, and not the Sal-Meter core signal track.

## Allowed public materials

Public repository materials may include:

- synthetic data
- sample data
- schema examples
- toy examples
- documentation
- example code
- non-identifiable demonstration files

## Not allowed

Do not upload:

- raw biosignal recordings from identifiable humans
- participant names or direct identifiers
- consent forms containing personal information
- phone numbers, email addresses, resident numbers, addresses, or medical record identifiers
- raw video, audio, or face data from real sessions
- private session logs
- private health-related records
- internal lab data packages
- uncontrolled human-subject datasets

## Public sample rule

Public examples exist only to demonstrate structure.

They must not be used to imply:

- Sal-Meter validation
- CAIS compliance
- diagnostic capability
- therapeutic capability
- clinical interpretation
- certification status

## Storage rule

Raw human data, if collected in future internal work, must remain outside public GitHub repositories.

Use secure internal storage with controlled access, versioned metadata, consent tracking, and audit trail discipline.

## Boundary sentence

No raw human data belongs in this repository.
