# Current Operating Model

This document defines the current operating model of the SICS Human-State Proxy Benchmark Track.

This repository is a public technical helper surface.

It is not the Sal-Meter core signal track.

It is not a CAIS-compliant device implementation.

It is not a clinical, diagnostic, therapeutic, surveillance, employment, insurance, educational, legal, eligibility, or human-ranking system.

## Track separation

### Sal-Meter Core Track

The Sal-Meter Core Track asks whether a new molecular–electrochemical signal interface can produce stable, repeatable, auditable signal behavior.

Current core order:

1. External Layer-0
2. SICS Internal Phase 0
3. Phase 1
4. Phase 2a
5. Phase 2b
6. LOCK 1 / LOCK 2
7. Future SDK / broader opening

### Proxy Benchmark Track

The Proxy Benchmark Track prepares the comparison layer.

It uses existing proxy signals such as ECG, HRV, EDA, PPG, EEG, eye / gaze, behavioral timing, task events, simulator state, and AI feedback logs.

The proxy track supports later comparison.

It does not replace the core track.
