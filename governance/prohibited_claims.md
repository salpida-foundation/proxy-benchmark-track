# Prohibited Claims

This document defines prohibited claims for the SICS Human-State Proxy Benchmark Track.

This repository is a public technical helper surface.

It is not the canonical authority layer.

It is not the Sal-Meter core signal track.

It is not a CAIS-compliant device implementation.

It is not a clinical, diagnostic, therapeutic, surveillance, employment, insurance, educational, legal, eligibility, certification, or human-ranking system.

---

## 1. Purpose

The purpose of this document is to prevent public claim drift.

The SICS Human-State Proxy Benchmark Track is useful only if it remains bounded.

It may support synchronized multimodal benchmark design, metadata discipline, leakage-safe evaluation, synthetic/sample data structure, dashboard mockups, closed-loop demo-lite planning, and future comparison with Sal-Meter I/G-channel inputs.

It must not be described as Sal-Meter.

It must not be described as CAIS compliance.

It must not be described as clinical, diagnostic, therapeutic, certified, validated, or commercial-device-ready.

Claims must never run ahead of evidence.

No person, contributor, laboratory, company, institution, repository fork, publication, dashboard, model, dataset, implementation, or derivative project may claim that this proxy benchmark track is:

- Sal-Meter

- CAIS-compliant

This repository prepares a benchmark support layer.
