# Claims Boundary

This document defines the claims boundary for the SICS Human-State Proxy Benchmark Track public technical helper repository.

This repository is a research-stage public helper surface.

It is not the canonical authority layer.

It is not the Sal-Meter core signal track.

It is not a CAIS-compliant device implementation.

It is not a clinical, diagnostic, therapeutic, surveillance, employment, insurance, educational, legal, eligibility, certification, or human-ranking system.

---

## 1. Purpose

The purpose of this document is to define what this repository may and may not claim.

The SICS Human-State Proxy Benchmark Track exists to prepare a bounded proxy benchmark support layer for Human-State-Aware AI Interaction.

It may support:

- synchronized multimodal benchmark design
- proxy signal metadata discipline
- event marker structure
- leakage-safe evaluation
- synthetic sample data structure
- baseline model skeletons
- dashboard mockups
- closed-loop demo-lite planning
- future comparison with Sal-Meter I/G-channel inputs

It must not claim:

- Sal-Meter status
- CAIS compliance
- clinical validity
- diagnostic validity
- therapeutic effect
- certification status
- commercial-device validation
- human-ranking authority
- surveillance scoring authority
- completed Sal-Meter validation

The repository becomes stronger by keeping its claims smaller, cleaner, and auditable.
