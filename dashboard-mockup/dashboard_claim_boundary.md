# Dashboard Claim Boundary

This document defines prohibited and allowed dashboard claim language for the **SICS Human-State Proxy Benchmark Track**.

It is a public helper boundary document.

It is not the Sal-Meter core signal track.

It does not define CAIS.

It does not grant CAIS compliance.

It does not validate Sal-Meter.

It does not validate benchmark performance.

It does not introduce raw human data.

It does not create diagnostic, clinical, therapeutic, surveillance, employment, insurance, educational, legal, eligibility, certification, or human-ranking authority.

---

## Purpose

Dashboard views can make uncertain things look final.

A field can become a score.

A score can become a verdict.

A chart can become a false instrument.

A color can become an accusation.

This document exists to prevent dashboard mockups from becoming overclaiming surfaces.

The dashboard may show structure.

The dashboard must not create authority.

---

## Core rule

A dashboard mockup may display synthetic/sample proxy benchmark fields.

A dashboard mockup must not imply:

```text
benchmark validation
scientific validation
Sal-Meter validation
CAIS compliance
clinical interpretation
diagnostic output
therapeutic feedback
surveillance readiness
certification readiness
human ranking
human truth scoring
```

Correct sentence:

```text
This dashboard mockup displays synthetic/sample proxy benchmark fields for structure demonstration.
```

Incorrect sentence:

```text
This dashboard measures human state.
```

## Review checklist

Before publishing any dashboard mockup, check:

```text
[ ] Is the dashboard clearly marked as research-stage?
[ ] Is the dashboard clearly marked as synthetic/sample when public?
[ ] Is it clear that this is not Sal-Meter?
[ ] Is it clear that this does not define CAIS?
[ ] Is it clear that this does not grant CAIS compliance?
[ ] Is it clear that this does not validate benchmark performance?
[ ] Is diagnostic language absent?
[ ] Is clinical language absent?
[ ] Is therapeutic language absent?
[ ] Is surveillance language absent?
[ ] Is certification language absent?
[ ] Is human-ranking language absent?
[ ] Are scores named as proxy/helper fields only?
[ ] Are colors and badges non-authoritative?
[ ] Are alerts framed as review prompts, not action commands?
[ ] Are dyadic or mediation references bounded, if present?
[ ] Are future Sal-Meter A/B comparison placeholders clearly marked as future only?
[ ] Are raw human data excluded?
[ ] Are identifiable data excluded?
[ ] Are clinical data excluded?
```

If any item fails, hold the dashboard.
