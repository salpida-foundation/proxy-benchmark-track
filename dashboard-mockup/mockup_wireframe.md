# Dashboard Mockup Wireframe

This document provides a text-only dashboard wireframe for the **SICS Human-State Proxy Benchmark Track**.

It is a public helper mockup document.

It is not the Sal-Meter core signal track.

It does not define CAIS.

It does not grant CAIS compliance.

It does not validate Sal-Meter.

It does not validate benchmark performance.

It does not introduce raw human data.

It does not create diagnostic, clinical, therapeutic, surveillance, employment, insurance, educational, legal, eligibility, certification, or human-ranking authority.

---

## Purpose

A dashboard is not neutral.

A dashboard can make uncertain fields look official.

A dashboard can turn a proxy field into a person score.

A dashboard can turn a mockup into a false instrument.

This wireframe exists to show how a future dashboard may remain bounded.

It shows layout language.

It does not show evidence.

It shows visual structure.

It does not show validation.
